Mr Gee: Hi everyone! Welcome to the Home Doctors, the show that looks at your home and makes it better.
Mr Gee: With me today is Patty. Patty, how can we help?
Patty: Thanks, Mr Gee! Well...
Patty: ...last week my husband came back late and it was so dark that he couldn’t find the light switch at the door.
Patty: Because of that, he tripped and fell.
Mr Gee: Oh dear.
Patty: After that happened, we installed a motion sensor that would switch on the light when it detects movement at night...
Patty: ...but I don’t think it’s programmed properly. The light comes on even when it's not needed.
Mr Gee: Don’t worry, Patty! Here, let the Home Doctors fix it for you.
That’s correct! No one is at the door, so the light is not needed.
As one of the Home Doctors, help Patty reprogram the motion sensor by choosing the correct outputs.
Each input is described here. Each input can be either 0 or 1.
01
The output is described here. The output can be either 0 or 1.
Move your mouse over each row of the truth table to see what each set of inputs represents.
Use the dropdown menu to choose an output for each set of inputs.
For instance, in the first row, an output of 0 means that the light should be off when no motion is detected at night.
Click the Check button to see if your answer is correct. The message will also explain why your answer is correct or wrong.
This concludes the tutorial. Now, help Patty reprogram the motion sensor so the light turns on only when needed!
Sometime later...
Mr Gee: All done! How do you like it, Patty?
Patty: Thanks, Mr. Gee! The light doesn’t turn on during the day anymore.
Patty: However, we’ve noticed a new problem.
Mr Gee: What is it?
Patty: Sometimes, when it’s cloudy during the day, I may want to switch on the light.
Patty: However, the old light switch doesn’t work anymore.
